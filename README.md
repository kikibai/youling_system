##  未来规划与核心概念
### 1、炒Agent的概念
* RAG从被动获取变成主动搜索。
* 主动规划路线
* 个性化“全过程的旅游陪伴”，记忆管理，自动游迹，信息整合：抖音，小红书，携程，大众点评，美团等平台

### 2、前端展示更新
* 类似高德的路径规划
* 知识图谱展示优化，热门景点，小众宝藏，两个图谱

### - Agent
* **Tools**：Agent的能力（都能干什么）
* （预算，路程远近，口味）我在一个地方看附近吃什么，->查看各平台推荐系统输出的topk，去b站，抖音评论区，交叉验证，最终得到决策。
* Agent对于用户画像 -> `setUserImage`
* Agent对于各平台的搜索 -> `CallMiniAgent(plan)`
* `sumerize`
* 大小Agent
